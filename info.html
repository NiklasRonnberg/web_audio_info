Web Audio API

Web Audio API is a powerful web-based interface for creating, processing, and playing audio directly in the browser. It enables synthesis, effects, filtering, mixing, and spatial audio positioning through a node-based system called an audio graph.

All audio processing happens inside an AudioContext, which provides precise timing, routing, and control over all connected audio nodes.

The API is designed for real-time interactive systems such as music tools, generative audio, games, and sonification systems.

The AudioContext is the central object in the Web Audio API. It acts as the runtime environment in which all audio nodes exist and are connected.

Without it, no audio can be generated or played.

It is typically created after a user interaction:

const audioContext = new AudioContext();

An Web Audio Example

The figure below shows the structure of a detuned synthesizer built using the Web Audio API, that is used in the traffic load sonification.

Detuned Synthesizer Example

The synthesizer uses two harmonic tones: 261.63 Hz and 392 Hz. Each tone is generated using three triangle wave oscillators.

Slight detuning between oscillators creates interference patterns (beating), which thickens the sound.

A shared gain node controls overall amplitude and is smoothly ramped to avoid clicks.

The second tone is dynamically shifted using a tone2Detune parameter, allowing real-time pitch modulation.

All oscillators are connected to a single output chain:

oscillators → gain → speakers

Real-Time Control

Parameters such as gain, detune spread, and pitch offset are continuously updated using smooth interpolation.

This avoids clicks and allows expressive modulation in real time.
